# Лабораторная работа №2

Однослойный перцептрон.

## Структура

- `main.py` — запуск работы и экспериментов
- `perceptron.py` — реализация однослойного перцептрона
- `data.py` — подготовка и генерация данных
- `metrics.py` — метрики качества и вывод таблиц
- `visualization.py` — графики
- `experiments.py` — эксперименты
- `notebooks/lab_2.ipynb` — исходный ноутбук

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
